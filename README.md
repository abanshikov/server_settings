# Settings server Debian/Ubuntu

1. Installing
```bash
$ sudo apt-get install wget
$ sh -c "$(wget https://raw.githubusercontent.com/abanshikov/server_settings/main/install.sh -O -)"
```

2. Install pluggins for vim
```bash
$ vim ~/.vimrc

:PlugInstall
```

3. Update settings zsh
```bash
$ source ~/.zshrc
```
